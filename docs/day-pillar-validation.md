# 日柱基準驗證

本文件記錄第一版日柱基準與後續人工驗證欄位。核心目前不因本文件而改變計算規則。

## 目前基準

- 日柱基準：`2000-01-01` 為 `戊午` 日。
- 推算方式：以基準日為起點，依 60 甲子循環逐日推算。
- 換日規則：第一版採 `23:00` 起換日。
- 時柱規則：依日干起子時，並依輸入的本地手錶時間判斷時辰。

## 待確認事項

- 需以多個年份的權威萬年曆資料驗證日柱循環。
- 需以節氣交界前後案例確認年柱、月柱與日柱同時穩定。
- 需以 22:59 / 23:00 / 00:xx 子時交界案例確認換日規則。
- 若未來支援不同換日流派，需把日柱與時柱案例拆成規則版本化測試。
