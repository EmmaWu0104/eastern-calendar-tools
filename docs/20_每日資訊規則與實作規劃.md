# 每日資訊規則與實作規劃

## 一、功能背景

每日資訊第一版用於補充「日子層級」的快速判斷資訊。

第一版包含：

1. 依日支五行推算衣著顏色。
2. 依日支六衝顯示衝煞生肖。
3. 依日支與年支六衝判斷歲破日。
4. 依 24 節氣前一日判斷四離四絕日。
5. 依節氣季節與日柱判斷天赦日。
6. 依夏至、立秋與庚日序判斷三伏日。

本階段不加入：

- 其他神煞。
- 宜忌斷語。
- 吉凶斷語。

## 二、日支五行與衣著顏色規則

### 地支五行

- 亥、子：水
- 寅、卯：木
- 巳、午：火
- 申、酉：金
- 辰、戌、丑、未：土

### 五行顏色

- 土：黃色、咖啡色
- 金：金色、銀色、白色、淺灰色
- 水：黑色、深藍色、深灰色
- 木：綠色
- 火：紅色、粉紅色、橘色、紫色

### 衣著判斷規則

- 讚的：日支五行所生之五行。
- 吉的：日支五行本身。
- 大忌：生日支五行者。

五行相生：

- 木生火
- 火生土
- 土生金
- 金生水
- 水生木

### 範例

若今天是丙午日：

- 日支為午。
- 午屬火。
- 火生土，所以「讚的」為土色。
- 同火，所以「吉的」為火色。
- 木生火，所以「大忌」為木色。

顯示文字：

```text
衣著：
讚的 土（黃色、咖啡色）
吉的 火（紅色、粉紅色、橘色、紫色）
大忌 木（綠色）
```

## 三、衝煞生肖規則

衝煞依日支六衝判斷。

地支六衝：

- 子午沖
- 丑未沖
- 寅申沖
- 卯酉沖
- 辰戌沖
- 巳亥沖

生肖對照：

- 子：鼠
- 丑：牛
- 寅：虎
- 卯：兔
- 辰：龍
- 巳：蛇
- 午：馬
- 未：羊
- 申：猴
- 酉：雞
- 戌：狗
- 亥：豬

範例：

午日：

- 午沖子。
- 子為鼠。
- 顯示：`衝煞：鼠`

## 四、歲破日規則

歲破日依年支與日支六衝判斷。

規則：

- 若日支沖年支，則為歲破日。
- 年支沿用本工具年柱規則：以立春切換。
- 日支沿用本工具日柱規則：23:00 換日。
- 不另建農曆年或民用年規則。

範例：

丙午年遇壬子日：

- 年支為午。
- 日支為子。
- 子午沖。
- 判定為歲破日。

UI 顯示：

- 是歲破日才顯示 `歲破日`。
- 平常日不顯示 `歲破：否`。

## 五、四離四絕日規則

四離四絕依 24 節氣前一日判斷，屬於日子資訊，第一版顯示在日柱卡片下方。

### 四離日

- 春分前一日：離日：木離日
- 夏至前一日：離日：火離日
- 秋分前一日：離日：金離日
- 冬至前一日：離日：水離日

### 四絕日

- 立春前一日：絕日：木旺水絕
- 立夏前一日：絕日：火旺木絕
- 立秋前一日：絕日：金旺土絕
- 立冬前一日：絕日：水旺金絕

### 判斷注意

- 第一版建議跟隨本工具既有日柱 23:00 換日規則。
- 也就是「前一日」應以本工具有效日判斷，而不是單純用民用日期 00:00 切換。
- 不使用真太陽時。
- 不另做每日通勝表查表。

## 六、天赦日規則

天赦日依節氣季節與日柱判斷。

### 季節區間

- 春季：立春至立夏前。
- 夏季：立夏至立秋前。
- 秋季：立秋至立冬前。
- 冬季：立冬至次年立春前。

### 判斷公式

- 春季逢戊寅日：天赦日。
- 夏季逢甲午日：天赦日。
- 秋季逢戊申日：天赦日。
- 冬季逢甲子日：天赦日。

### 時間規則

- 季節區間沿用本工具既有 24 節氣時間。
- 日柱沿用本工具 23:00 換日規則。
- 冬季需支援跨年判斷。
- 不使用真太陽時。
- 不查每日通勝表。

### UI 顯示

- 是天赦日才顯示 `天赦日`。
- 平常日不顯示 `天赦：否`。
- 顯示在日柱卡片下方，與歲破日、四離四絕日同層。

## 七、三伏日規則

三伏日依夏至、立秋與庚日序判斷。

### 第一版規則

- 初伏：夏至當日起算第 3 個庚日。
- 中伏：夏至當日起算第 4 個庚日。
- 末伏：立秋當日起算第 1 個庚日。

### 庚日定義

- 日柱天干為「庚」之日。
- 日柱沿用本工具 23:00 換日規則。
- 夏至與立秋時間沿用本工具既有 24 節氣時間。
- 「當日起算」表示若夏至或立秋所在的有效日本身為庚日，該日可列入第 1 個庚日。
- 不使用真太陽時。
- 不查每日通勝表。

### 特殊規則

- 若中伏與末伏落在同一個庚日，第一版採用「提前十日補一伏」規則。
- 此規則屬本工具第一版採用版本，後續若要對照通勝，需另開校驗資料與案例。

### UI 顯示

- 是初伏日顯示 `初伏`。
- 是中伏日顯示 `中伏`。
- 是末伏日顯示 `末伏`。
- 平常日不顯示三伏資訊。
- 顯示在日柱卡片下方，與歲破日、天赦日、四離四絕日同層。

## 八、UI 規劃

### 衣著顏色顯示位置

顯示在查詢區的「查詢日」下一行。

畫面格式：

```text
查詢日：星期二
衣著：
讚的 土（黃色、咖啡色）
吉的 火（紅色、粉紅色、橘色、紫色）
大忌 木（綠色）
```

### 衝煞、歲破、天赦、三伏、四離四絕顯示位置

顯示在四柱區塊的「日柱」卡片下方，接在建除十二神下方。

格式範例：

```text
建除：破日
衝煞：鼠
歲破日
天赦日
初伏
離日：木離日
```

若不是歲破日，不顯示歲破行。

若不是天赦日，不顯示天赦行。

若不是三伏日，不顯示三伏行。

三伏日只在符合初伏、中伏、末伏當日顯示。

若不是四離四絕日，不顯示離日 / 絕日行。

若是絕日：

```text
建除：破日
衝煞：鼠
絕日：木旺水絕
```

### 月破暫不另行顯示

原因：

- 月破可由建除十二神的「破日」表現。
- 第一版不額外顯示「月破日」，避免日柱卡片資訊過重。
