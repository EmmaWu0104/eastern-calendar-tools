* `data/1080.md` 是奇門 1080 盤母資料 / canonical source。
* `data/qimen/plates/**` 是由 `data/1080.md` 產生的正式衍生資料。
* 若盤面內容錯誤，優先修 `data/1080.md`，再重跑 parser / converter / adapter / writer。
* 不建議直接手修 `data/qimen/plates/**`。
* 若是 schema / JSON 結構問題，應修 converter / adapter / writer。
