# 干支曆核心規則

本文件記錄第一版干支曆核心採用的計算規則。第一版使用使用者輸入的本地手錶時間。

## 年柱

- 年柱以立春作為干支年切換點。
- 輸入時間若早於當年立春，年柱使用前一個干支年。
- 干支年序列以 1984 年立春後為甲子年推算。

## 月柱

- 月柱以 12 個「節」切換月令，不使用中氣切月。
- 月令對照如下：

| 節 | 月令地支 |
| --- | --- |
| 立春 | 寅 |
| 驚蟄 | 卯 |
| 清明 | 辰 |
| 立夏 | 巳 |
| 芒種 | 午 |
| 小暑 | 未 |
| 立秋 | 申 |
| 白露 | 酉 |
| 寒露 | 戌 |
| 立冬 | 亥 |
| 大雪 | 子 |
| 小寒 | 丑 |

- 月干依年干起寅月：
  - 甲己年：丙寅起
  - 乙庚年：戊寅起
  - 丙辛年：庚寅起
  - 丁壬年：壬寅起
  - 戊癸年：甲寅起

## 日柱

- 第一版採用明確基準日常數：`2000-01-01` 為 `戊午` 日。
- 此基準來自常見 Julian Day 干支日公式可推得的結果，第一版先作為固定常數使用。
- 後續需以權威萬年曆或多來源日柱資料再次交叉驗證。
- 日柱基準是第一版最重要的驗證點之一，驗證紀錄見 [day-pillar-validation.md](./day-pillar-validation.md)。

## 子時換日

- 第一版採用 23:00 起換日。
- 也就是 `22:59:59` 仍屬當日干支日，`23:00:00` 起採下一日干支日。
- UI 與回傳 `ruleNotes` 皆明確標示此規則。

## 真太陽時

- 第一版使用使用者輸入的本地手錶時間。
- `datetime-local` 不含時區資訊。
- 預設值依使用者瀏覽器所在地本機時間顯示。
- 節氣資料保留 UTC / Asia-Taipei 欄位，但輸入時間第一版依瀏覽器本機時區解析。
- 第一版不套用真太陽時。
- 第一版不做出生地、經緯度或均時差校正。
- 後續若要支援跨時區出生地排盤，需新增時區、地點或經度、均時差與真太陽時校正。
- 校正後若跨時辰、跨日或跨節氣，需重新判斷四柱。

## 時柱

時辰地支：

| 時間 | 地支 |
| --- | --- |
| 23:00-00:59 | 子 |
| 01:00-02:59 | 丑 |
| 03:00-04:59 | 寅 |
| 05:00-06:59 | 卯 |
| 07:00-08:59 | 辰 |
| 09:00-10:59 | 巳 |
| 11:00-12:59 | 午 |
| 13:00-14:59 | 未 |
| 15:00-16:59 | 申 |
| 17:00-18:59 | 酉 |
| 19:00-20:59 | 戌 |
| 21:00-22:59 | 亥 |

時干依日干起子時：

- 甲己日：甲子起
- 乙庚日：丙子起
- 丙辛日：戊子起
- 丁壬日：庚子起
- 戊癸日：壬子起

## 待確認事項

- `2000-01-01 戊午日` 基準日需以權威萬年曆交叉驗證。
- 23:00 起換日為第一版預設，後續若支援流派選項，需允許切換子初或子正換日。
- 節氣資料若重新產生，需重新執行邊界案例驗證。
- 後續若支援跨時區出生地排盤，需另行設計時區與真太陽時 API。
