## 奇門狀態規則

- 所有狀態判斷一律使用奇門有效日。
- 奇門有效日於 23:00 換日；23:00 之後算隔天。
- 正授僅在交節氣當天的奇門有效日，其日干支為甲子、甲午、己卯或己酉時成立。
- 「實際節氣 == 奇門節氣」不是正授的充分條件。
- 超神表示符頭已到，但節氣尚未交接。
- 接氣表示節氣已交接，但符頭尚未到。
- 置閏維持既有置閏 timeline 與置閏優先序；UI 不顯示 full cycle draft 的內部診斷備註。

## 術語與星名

- 現行奇門狀態名稱使用「正授」，不以「正常」表示奇門狀態。
- canonical 星名使用「天衝」。
- code 可保留「天沖」作為 alias／normalize 防呆；正式資料與 UI 一律使用「天衝」。

## 奇門資訊卡

目前資訊卡固定顯示：

- 節氣：實際節氣。
- 起局：奇門節氣 + 元別 + 遁別 + 局數 + 狀態。
- 時辰：日柱 + 時柱。
- 值符星：星名 + 宮位。
- 值使門：門名 + 宮位。
- 天乙星：星名 + 宮位。

實際節氣與奇門節氣是不同概念：實際節氣固定顯示於「節氣」，起局則使用奇門節氣。盤面上方原有的遁別／局數／時柱／直符／直使舊摘要行已移除，避免與資訊卡重複。

- 中宮地盤干寄到坤宮，顯示於坤宮地盤干旁邊。

- 此調整避免天地盤干區域擁擠，並為後續格局顯示保留空間。

## 月曆與十二時辰查詢器

- 查詢器已從以 `datetime-local` 為主的操作方式，改為月曆加十二時辰。
- 月曆以星期一為每週第一天，標題順序為一、二、三、四、五、六、日。
- 可查詢年份範圍為 1900～2100。
- 今天與選取日皆有標記。
- 切換月份不會立即改變查詢日，必須點選日期才會查詢。
- 手動選日期或時辰會暫停 auto now；點「現在時間」會恢復 auto now。

## 十二時辰規則

時辰按鈕以時辰起點作為查詢時間：

| 時辰 | 查詢起點 |
| --- | --- |
| 子 | 23:00 |
| 丑 | 01:00 |
| 寅 | 03:00 |
| 卯 | 05:00 |
| 辰 | 07:00 |
| 巳 | 09:00 |
| 午 | 11:00 |
| 未 | 13:00 |
| 申 | 15:00 |
| 酉 | 17:00 |
| 戌 | 19:00 |
| 亥 | 21:00 |

- 整點整分即進入新時辰：08:59 仍為辰、09:00 為巳、22:59 仍為亥、23:00 為子。
- 選擇子時時，會使用當日 23:00。例如點選 2026/07/19 子時，實際查詢時間為 `2026-07-19T23:00`，不會轉成 `2026-07-19T00:00`。
- 此行為與專案的 23:00 換日規則一致。
