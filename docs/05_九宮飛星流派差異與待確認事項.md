# 九宮飛星流派差異與待確認事項

## 第一版採用範圍

- 本專案只做時間飛星，不做陽宅飛星。
- 第一版只計算運盤、年盤、月盤、日盤、時盤。
- 運盤、年盤、月盤一律順飛。
- 日盤、時盤依節氣系統決定順飛或逆飛。
- 本專案採用使用者指定規則，其他流派不納入第一版。

## 不納入第一版的內容

- 陽宅盤。
- 山星、向星。
- 宅運盤、坐向盤。
- 替卦、兼線、元龍。
- 陽宅斷語。
- 真太陽時。
- 出生地、經緯度、均時差校正。

## 流派差異

九宮飛星在不同資料、師承或軟體中，常見差異包含：

- 日盤與時盤是否以節氣系統決定順逆。
- 日盤六組節氣的分界與入中表。
- 時盤日支分組與子時起星表。
- 是否採 23:00 起換日。
- 是否使用真太陽時。
- 年盤是否嚴格以立春切換。
- 月盤是否只以 12 節切月。

以上差異第一版不做選項化；目前固定採用使用者指定規則。

## 待確認事項

- 日盤六組節氣入中表需以權威來源或人工案例交叉驗證。
- 時盤完整查表資料需以使用者提供表格或權威來源再逐列核對。
- 夏至前後、冬至前後的時盤系統切換需增加更多邊界測試。
- 23:00 起換日時，日盤與時盤是否皆採切換後日柱 / 日支，需持續以案例確認。
- 若未來加入其他流派，需在資料結構中加入 `ruleSet` 或 `school` 欄位，避免不同規則混用。
