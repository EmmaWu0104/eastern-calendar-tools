# 金函玉鏡超神接氣規則整理

## 一、文件目的

金函玉鏡日盤目前已能以 `dayPillar + dunType` 查詢 60 甲子日 x 陽遁 / 陰遁，共 120 盤資料。主畫面 UI 目前仍由使用者手動選擇「陽遁 / 陰遁」。

本文件目的，是整理未來要自動判斷 `dunType` 時所需的超神接氣規則，並區分已確認規則、目前理解與待確認疑點。

本文件不是演算法實作文件。在規則未完全確認前，不應自動套用到正式 UI。

## 二、目前原文

如遇冬至、夏至，以甲日換順逆，如甲在冬至前七日，仍依夏至後逆行，冬至後三日亦必逆行，須到第四逢之甲，方用冬至後順行之例，名曰「超神」，謂中氣已到，六甲一旬十日，超過中氣也。

如甲子日在冬至前四日，即依冬至後順行，名曰「接氣」，謂中氣未到，六甲旬頭先到，迎接中氣也。

## 三、已確認規則

1. 金函玉鏡陰陽遁切換與冬至、夏至有關。
2. 金函玉鏡不是單純在冬至瞬間切陽遁、夏至瞬間切陰遁。
3. 原文明確說「以甲日換順逆」。
4. 冬至後理論方向為順行，可對應目前資料中的「陽遁」。
5. 夏至後理論方向為逆行，可對應目前資料中的「陰遁」。
6. 若甲日在冬至前七日，冬至前仍依夏至後逆行。
7. 即使冬至已到，冬至後三日仍可能繼續逆行。
8. 須到「第四逢之甲」，才使用冬至後順行。
9. 若甲子日在冬至前四日，可提前依冬至後順行，稱為接氣。
10. 因此未來自動判斷 `dunType` 必須獨立於一般干支曆與九宮飛星規則。

## 四、目前理解

以下為目前理解，非最終演算法。

- 一般直覺會以冬至 / 夏至作為陰陽遁分界。
- 但金函玉鏡原文指出，實際換順逆要看「甲日」與冬至 / 夏至的相對位置。
- 「超神」可理解為：中氣已到，但六甲旬頭與中氣錯位，因此不立即換遁。
- 「接氣」可理解為：中氣未到，但六甲旬頭已接近中氣，因此提前換遁。
- 對 UI 而言，這會影響使用哪一張盤：
  - `dayPillar + 陽遁`
  - `dayPillar + 陰遁`

## 五、待確認疑點

以下疑點禁止自行推導，需等待資料來源、人工判例或權威規則確認。

1. 「甲日」是指所有天干為甲的日，還是特指甲子旬頭？
2. 原文第二段寫「甲子日在冬至前四日」，是否只是舉例，或接氣必須特指甲子日？
3. 「第四逢之甲」精確意義為何？
   - 是從哪個甲日起算？
   - 是否包含冬至前的甲日？
   - 是第四個甲日，還是第四日逢甲？
4. 冬至規則是否完全對稱套用到夏至？
5. 夏至前後是否也有相同的七日 / 三日 / 四日界線？
6. 超神 / 接氣是否以「日」為單位，還是要精確到節氣交接時刻？
7. 若節氣交接發生在一天中間，該日是否整日採同一遁？
8. 與目前干支曆 23:00 換日規則如何協調？
9. 若未來加入真太陽時，是否需用校正後時間重新判斷節氣邊界與日柱？
10. 是否有其他版本原文或不同流派判法？
