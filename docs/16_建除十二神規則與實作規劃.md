# 建除十二神規則與實作規劃

## 一、功能背景

建除十二神為每日值日資訊，常見名稱依序為建、除、滿、平、定、執、破、危、成、收、開、閉。

本工具第一版只計算建除十二神名稱，用於顯示每日值日狀態；不做宜忌、不做吉凶斷語，也不套用每日通勝表。

## 二、第一版規則

第一版建除十二神使用目前月令月支與日柱地支計算。

- 月支沿用本工具既有月令規則：依 12 節切換。
- 日支沿用本工具既有日柱規則：23:00 換日。
- 不查每日通勝表。
- 不使用真太陽時。
- 不影響 72 候、金函玉鏡日盤、九宮飛星。

因此，建除十二神會隨本工具既有干支曆核心輸出的月令與日柱變化，不另建一套時間規則。

## 三、建除十二神順序

建除十二神順序如下：

1. 建
2. 除
3. 滿
4. 平
5. 定
6. 執
7. 破
8. 危
9. 成
10. 收
11. 開
12. 閉

## 四、地支序號

地支序號採 0 到 11：

| 地支 | 序號 |
| --- | ---: |
| 子 | 0 |
| 丑 | 1 |
| 寅 | 2 |
| 卯 | 3 |
| 辰 | 4 |
| 巳 | 5 |
| 午 | 6 |
| 未 | 7 |
| 申 | 8 |
| 酉 | 9 |
| 戌 | 10 |
| 亥 | 11 |

## 五、計算公式

```text
建除序號 = (日支序號 - 月支序號 + 12) % 12
```

計算出的建除序號對應「建、除、滿、平、定、執、破、危、成、收、開、閉」順序。

## 六、範例

- 巳月巳日：`(5 - 5 + 12) % 12 = 0`，為建日。
- 巳月午日：`(6 - 5 + 12) % 12 = 1`，為除日。
- 巳月亥日：`(11 - 5 + 12) % 12 = 6`，為破日。

## 七、UI 規劃

建除十二神雖然依月令計算，但呈現上屬於每日資訊。

第一版建議顯示在四柱區塊的「日柱」卡片下方，格式例如：

```text
建除：破日
```

不建議放在「節氣與月令」區塊，避免該區塊承載過多資訊，也避免使用者誤以為建除十二神是節氣本身的附屬資料。

## 八、後續分包

- 第 17 包：新增建除十二神計算模組與測試。
- 第 18 包：在日柱卡片下方顯示建除十二神。

## 九、限制與注意事項

- 第一版不修改干支曆核心規則。
- 第一版不修改 72 候判斷。
- 第一版不修改金函玉鏡日盤。
- 第一版不修改九宮飛星。
- 第一版不加入宜忌、吉凶、旺衰或斷語。
- 若後續要加入每日通勝表，需另開資料來源與校驗流程，不應覆蓋本公式版建除計算。
