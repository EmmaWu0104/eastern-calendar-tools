# 九宮飛星規則設計草案

本文件記錄第一版九宮飛星的時間盤規則。第一版只做「時間飛星」，不做陽宅飛星。

## 範圍

- 只計算當下時間的運盤、年盤、月盤、日盤、時盤。
- 不做陽宅盤。
- 不做山星、向星、宅運盤、坐向盤、替卦、兼線、元龍、陽宅斷語。
- 不加入真太陽時。
- 暫時不接 UI。
- 本專案採用使用者指定規則，其他流派不納入第一版。

## 飛星順序

順飛：

```text
中 -> 乾 -> 兌 -> 艮 -> 離 -> 坎 -> 坤 -> 震 -> 巽
```

逆飛：

```text
中 -> 巽 -> 震 -> 坤 -> 坎 -> 離 -> 艮 -> 兌 -> 乾
```

## 各盤規則

### 運盤

- 依三元九運。
- 幾運即幾星入中。
- 一律順飛。
- `2024-2043` 為九運，九入中宮。

### 年盤

- 立春換年。
- 使用干支曆核心已計算好的有效年份。
- 入中公式：`11 - 西元年份各位數字和`，結果 normalize 到 `1-9`。
- 一律順飛。
- `2026` 丙午年為一白入中宮。

### 月盤

- 依節換月。
- 使用干支曆核心已計算好的年支與月支。
- 年支分三組：`子午卯酉`、`辰戌丑未`、`寅申巳亥`。
- 一律順飛。
- `2026` 丙午年巳月，月盤五入中宮。

### 日盤

- 使用 23:00 換日後的日柱。
- 依目前所在節氣歸類為六組，決定順飛或逆飛。
- 冬至順、雨水順、穀雨順為順飛。
- 夏至逆、處暑逆、霜降逆為逆飛。
- `2026/05/29` 癸卯日，小滿，日盤七入中宮，順飛。

### 時盤

- 使用 23:00 換日後的日支。
- 使用目前時支。
- 尚未進入夏至系統時使用冬至順表。
- 進入夏至系統後使用夏至逆表。
- 時盤表完整展開成 `dayBranch + hourBranch` 查表資料，不自行推導簡化。
- `2026/05/29` 癸卯日，巳時，小滿，時盤六入中宮，順飛。
