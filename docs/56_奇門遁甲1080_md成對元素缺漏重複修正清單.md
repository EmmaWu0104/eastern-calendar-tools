# 56 奇門遁甲 1080.md 成對元素缺漏重複修正清單

本文件是新第 94 包 docs-only 清單，目的不是修 `data/1080.md`，而是重跑 `qimen1080MarkdownParser` 後，將剩餘 `MISSING_*` / `DUPLICATE_*` diagnostics 依「同一盤」分組，提供後續人工逐筆對照手修。

本包不修改 `data/1080.md`、parser、tests、UI，不產生 JSON，也不寫入 `data/qimen/plates/**`。

## 1. 本包目的

本包整理下列成對元素缺漏 / 重複 diagnostics：

* `MISSING_DOOR` / `DUPLICATE_DOOR`
* `MISSING_STAR` / `DUPLICATE_STAR`
* `MISSING_HEAVEN_STEM` / `DUPLICATE_HEAVEN_STEM`
* `MISSING_EARTH_STEM` / `DUPLICATE_EARTH_STEM`
* `MISSING_DEITY` / `DUPLICATE_DEITY`

清單以同一盤為單位列出遁別、局數、日群、時柱、涉及 code、missing 內容、duplicate 內容、duplicate 對應 raw cells，以及建議人工判讀方向。

## 5. 修正總原則

* 每一類都要成對看。
* 不要只看到 missing 就補字。
* 不要只看到 duplicate 就刪字。
* 優先找同一盤內「少 A、多 B」的對應關係。
* 外宮格式仍為：天盤干 + 九星 / 地盤干 + 八門 + 八神。
* 中宮 / 天禽 / 寄宮不要用外宮規則硬改。
* 若 duplicate raw cells 包含 `center` 或 `天禽`，必須先判斷是否為寄宮規則，不要直接改中宮。
